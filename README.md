# Statistische Zeitreihenanalyse zur Absatzprognose im Modehandel

Dieses Repository enthält den vollständigen Python-Code zur Projektarbeit „Statistische Zeitreihenanalyse zur Absatzprognose im Modehandel – Eine modellgestützte Entscheidungsgrundlage für die Nachversorgung“.

## Beschreibung

Ziel der Arbeit ist die Untersuchung, wie geeignet statistische Zeitreihenmodelle (ARIMA, SARIMA, SARIMAX) zur Prognose von Verkaufsmengen verschiedener Produktkategorien in einem Modeunternehmen sind. Auf dieser Grundlage sollen Handlungsempfehlungen für die operative Nachversorgung abgeleitet werden.

Die Analyse basiert auf realen Verkaufsdaten aus dem B2C-Segment eines Modeunternehmens. Der gesamte Workflow orientiert sich am CRISP-DM-Modell (Cross Industry Standard Process for Data Mining).

## Inhalte des Notebooks

Datenaufbereitung: Extraktion, Bereinigung und Aggregation der Verkaufsdaten

Explorative Datenanalyse: Identifikation von Trends, Saisonalitäten und Ausreißern

Modellbildung: Implementierung von ARIMA-, SARIMA- und SARIMAX-Modellen mit Python

Evaluation: Vergleich der Modelle anhand statistischer Fehlermetriken (MAE, MSE, WMAPE, MBE)
